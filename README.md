## 💡 什么是 BoltUtil？

传统的支付网关通常充当“资金托管人”，而 BoltUtil 回归了区块链的本质。我们通过高性能节点实时监听区块链网络，确保资金流动透明、安全、直接。

### 核心工作逻辑：无感、直接、纯粹
1. **直达钱包**：资金不经过 BoltUtil，直接进入您的非托管钱包 (MetaMask, Trust Wallet 等)。
2. **智能监控**：亚秒级扫描目标地址，捕获每一笔链上波动。
3. **自动触发**：通过高性能 Webhook 和邮件系统，立即通知后端完成发货。
